# Badge Data Cleaning

This project focuses on **cleaning and organizing badge data** extracted from PDF files using Excel.  
The goal is to transform raw data into a structured format suitable for analysis.

## Dataset
- Source: `badge_data_extract.pdf`
- Cleaned version: `badge_data_cleaned.xlsx`

## Tools
- Microsoft Excel

## Steps Performed
1. Extract data from PDF.
2. Transform and structure data in Excel.
3. Remove duplicates, errors, and inconsistencies.
4. Prepare dataset for further analysis.

## Usage
- Open `badge_data_cleaned.xlsx` in Excel to explore the cleaned dataset.
